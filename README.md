Tâche d'évaluation data basée sur les données de compétition CommonLit - Evaluate Student Summaries de Kaggle

1. 对于同一个 Prompt，比较两个不同学生的 R。设定其中一个 R*为好的回答，需要对二者的相同点和不同点展开分析。
2. 评估结合内容：结合两个回答生成新回答，对其进行分析和分数评估。
